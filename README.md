# MiDiff: MIDI-Conditioned Diffusion Models for Drum Audio Enhancement

Anonymous project page for MiDiff: a MIDI-conditioned diffusion approach to drum audio enhancement (symbolic conditioning via early injection, FiLM, and cross-attention on an SGMSE-style backbone).

This copy is maintained **without author names, affiliations, personal links, or institution-specific metadata** for anonymous sharing (for example supplementary material under blind review).

## Abstract (short)

Drum audio enhancement using MIDI-aligned symbolic information to guide diffusion denoising, combining complementary conditioning strategies in one architecture. Reported metrics include Fréchet Audio Distance and perceptual evaluations on drum (clean, noisy, MIDI) triplets.

## Features

- Interactive audio comparisons and static spectrograms  
- MIDI piano-roll visualization with playback  
- Charts for quantitative comparisons  
- Static site: HTML, Bulma, JavaScript; spectrograms can be regenerated with the included Python script
